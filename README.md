# V2EX

V2EX的非官方Android客户端，极力遵循Material Design风格

### 实现功能
* 查看最新和最热话题
* 查看所有节点
* 查看节点里的话题
* 查看话题详情、评论
* 评论话题
* 发表新话题
* 登录
* 查看用户信息

> 依赖picasso、okhttp、retrofit    
> 有些地方参考了[v2ex-android](https://github.com/greatyao/v2ex-android)的实现
